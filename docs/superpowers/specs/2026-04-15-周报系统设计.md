# 周报系统设计文档

**日期**: 2026-04-15
**状态**: 已批准

---

## 一、系统概述

- **系统名称**: 每日工作记录与周报导出系统
- **系统类型**: 本地单用户网页应用
- **核心功能**: 记录每日工作任务，自动生成周报Excel
- **目标用户**: 普通职场用户（单人使用）

---

## 二、技术栈

- **后端**: Flask (Python)
- **数据库**: MySQL (localhost:3306, root/123456)
- **前端**: 纯HTML/JS (无框架)
- **Excel导出**: openpyxl

---

## 三、数据模型

### 3.1 任务表 (tasks)

```sql
CREATE TABLE tasks (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(200) NOT NULL,
    category VARCHAR(50) NOT NULL,
    priority VARCHAR(10) DEFAULT '中',
    status VARCHAR(20) DEFAULT '未开始',
    plan_start DATETIME,
    plan_end DATETIME,
    progress_note TEXT,
    pause_reason TEXT,
    sort_order INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 3.2 分类枚举

教学、竞赛、就业、科研、项目、职能组、院校支撑

### 3.3 状态枚举

未开始、进行中、已完成、暂停

### 3.4 优先级枚举

高、中、低

## 四、功能需求

### 4.1 任务管理

#### 4.1.1 新增任务
- 输入字段：任务名称、所属分类、优先级、计划开始时间、计划完成时间
- 必填校验：任务名称、所属分类、计划开始时间、计划完成时间
- 默认状态：未开始
- 默认优先级：中

#### 4.1.2 编辑任务
- 支持修改所有字段
- 操作便捷，3步以内完成

#### 4.1.3 删除任务
- 确认后删除

#### 4.1.4 更新状态
- 支持状态：未开始 → 进行中 → 已完成
- 支持状态：进行中 → 暂停（暂停时必填原因）
- 状态更新时可添加进度备注

#### 4.1.5 排序调整
- 支持在同一分类内调整任务显示顺序
- 上移/下移操作

### 4.2 每日任务查看

- 默认展示当日所有任务
- 支持按状态筛选（全部/进行中/已完成/暂停/未开始）
- 支持按优先级排序
- 支持切换查看历史日期任务
- 按分类分组展示

### 4.3 周报导出

#### 4.3.1 周次选择
- 按自然周选择（上周一至上周日）
- 自动计算上周周期

#### 4.3.2 周报内容抓取
- 抓取周期内状态为"已完成"的任务
- 按六大分类归类
- 每条任务展示：任务名称 + 完成进度备注

#### 4.3.3 周报基本信息（固定值）
- 汇报人：王弯弯
- 所属项目：河工大
- 汇报时间段：根据选择的周期自动填充
- 提交时间：导出时的时间

#### 4.3.4 周报模板格式

| 列 | 内容 |
|----|------|
| 工作分类 | 教学/竞赛/就业/科研/项目/职能组/院校支撑 |
| 本周计划 | 手动填写区域 |
| 本周完成情况 | 自动填充已完成任务（名称+备注） |
| 下周工作计划 | 手动填写区域 |
| 待协调、支撑事项 | 手动填写区域 |

#### 4.3.5 导出格式
- 仅支持Excel格式（.xlsx）
- 文件命名：`YYYYMMDD-YYYYMMDD-周报.xlsx`

## 五、页面结构

### 5.1 任务管理页 (/tasks)
- 任务列表，按分类分组
- 新增任务表单
- 编辑任务弹窗
- 状态更新按钮
- 排序调整按钮

### 5.2 每日任务页 (/daily)
- 日期选择器
- 当日任务列表
- 状态/分类筛选器
- 优先级排序

### 5.3 周报导出页 (/report)
- 周次选择器
- 周报预览区（只读）
- 导出Excel按钮

## 六、非功能需求

### 6.1 易用性
- 操作简单，新增任务、状态更新、导出周报均3步以内完成
- 界面简洁，重点信息突出

### 6.2 稳定性
- 数据存储安全
- 导出周报无卡顿

### 6.3 隐私性
- 个人工作记录仅本人可见

## 七、优先级

- **P1（必做）**: 任务新增、状态更新、每日任务查看、周报导出
- **P2（优化）**: 任务编辑、删除、排序调整、筛选功能
- **P3（可选）**: 数据备份、搜索功能、数据统计
